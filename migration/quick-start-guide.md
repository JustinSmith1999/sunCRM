# 🚀 Quick Start Migration Guide

## Step 1: Export from Salesforce (30 minutes)

### Data Export Wizard
1. **Login to Salesforce** → Setup → Data → Data Export
2. **Select "Export Now"** (or schedule for off-hours)
3. **Choose these objects** (check all):
   - ✅ Account
   - ✅ Contact  
   - ✅ Opportunity
   - ✅ Lead
   - ✅ Task
   - ✅ Event
   - ✅ Case
   - ✅ User
   - ✅ Product2
   - ✅ Quote
   - ✅ Campaign
4. **Select "Include all data"**
5. **Choose CSV format**
6. **Click "Start Export"**

### Configuration Export
1. **Object Manager** → For each object:
   - Export field definitions (screenshot or copy)
   - Note custom fields and their types
   - Document validation rules
   - Save page layout screenshots

2. **Process Automation**:
   - Export workflow rules
   - Document approval processes
   - Note email templates

## Step 2: Prepare Data (15 minutes)

### Organize Files
Create this folder structure:
```
salesforce-migration/
├── data/
│   ├── Account.csv
│   ├── Contact.csv
│   ├── Opportunity.csv
│   ├── Lead.csv
│   ├── Task.csv
│   ├── Case.csv
│   └── User.csv
├── config/
│   ├── custom-fields.xlsx
│   └── workflows.xlsx
└── reports/
    └── current-reports.xlsx
```

### Quick Data Review
- **Check record counts**: Note how many records per object
- **Identify custom fields**: List any custom fields you use heavily
- **Note integrations**: What systems connect to Salesforce?
